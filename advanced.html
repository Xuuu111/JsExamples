Explanations of Javascript code

setCookie

The parameters of the function above are the name of the cookie (cname), the value of the cookie (cvalue), and the number of days until the cookie should expire (exdays).

The function sets a cookie by adding together the cookiename, the cookie value, and the expires string.

getCookie
  • Take the cookiename as parameter (cname).
  • Create a variable (name) with the text to search for (cname + "=").Decode the cookie string, to handle cookies with special characters, e.g. '$'
  • Split document.cookie on semicolons into an array called ca (ca = decodedCookie.split(';')).
  • Loop through the ca array (i = 0; i < ca.length; i++), and read out each value c = ca[i]).
  • If the cookie is found (c.indexOf(name) == 0), return the value of the cookie (c.substring(name.length, c.length).
  • If the cookie is not found, return "".
checkCookie

Last, we create the function that checks if a cookie is set.

If the cookie is set it will display a greeting.

Matrix effect

Create a cool matrix effect with JavaScript.

Explanation of JavaScript code:

Firstly you need to get the canvas and context and make it fullscreen. use var c = document.getElementById("c"); var ctx = c.getContext("2d");. After that set the canvas to fullscreen with c.height = window.innerHeight; c.width = window.innerWidth;


Since we are using chinese characters falling down, we need to create a variable with them and then split them to make an array. use .split("") for the split.


var columns = c.width/font_size; sets the amount of columns to the approximate amount depending on the font size and canvas full size. drops = [] creates an array of drops and //x below is the x coordinate //1 = y co-ordi then sets the initial x and y coordinates.


The function draw() is the one that draws the canvas and characters. See the code comments for more detail on the style and drop effect. Lastly setInterval(draw, 33); is the codeline which runs the draw() function every 33ms
