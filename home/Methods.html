  • JavaScript has methods like string methods, date methods and number methods. There are also Object methods that are explained on this page on the functions page.

    Built-in JavaScript methods

    This is an example that uses the built-in method toLowerCase of the String object. It converts all capital letters to small letters.

    var message = "HELLO WORLD!";
    var x = message.toLowerCase();
    The value of x is then hello world!

    Adding new methods to an object is done insde the constructor function

    function car(type, color, horsepower,interiorColor){
this.type= type;
this.color= color;
this.horsepower = horsepower;
this.interiorColor = interiorColor;
this.changeHorsepower = function(breakhorsepower){
this.horsepower = breakhorsepower;
	};
}

    the changeHorsepower() function assigns the value of breakhorsepower to the car's horsepower property thus changing it to a new value
