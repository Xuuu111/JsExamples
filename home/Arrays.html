  • Array

  • JavaScript arrays work in the same ways as arrays do in other programming languages. Arrays are used to store mutiple values in a single variable. An Array can look something like this. 

    var guitars = ["Gibson", "Fender", "ESP"];

  • How do I create an array?

  • var array-name = ["item1", "item2", "item3",.......];

  • Spaces and line breaks do not break the code. The decleration can span mutiple lines. 

  • var guitars = [
     "Gibson",
     "Fender",
     "ESP"
    ];

    Do not put a comma after the last element like this 

    "ESP",

  • Accessing the elements of an Array

    • Accessing the elements of an Array is an fundamental skill to learn when dealing with Arrays.
    So how do I access the first element of my array?
    This statement accesses the first value of our array which is the element "Gibson" 

    var name = guitars[0];

    This statement modifies the second element "Fender" to "Ibanez"

    guitars[1]="Ibanez";
