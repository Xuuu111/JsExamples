Google maps

The functionality of the map is provided by a JavaScript library located at Google.

Explanations of Javascript code

The mapCanvas variable is the map's HTML element.

The mapOptions variable defines the properties for the map.

The center property specifies where to center the map (using latitude and longitude coordinates).

The zoom property specifies the zoom level for the map (try to experiment with the zoom level).

The google.maps.Map object is created with mapCanvas and mapOptions as parameters.

Snackbar

Snackbars are often used as tooltips/popups to show a message at the bottom of the screen.

Explanation of JavaScript code:

The snackbar has one function "myFunction()" which gets the element with var x = document.getElementById("snackbar"). it then adds a class called "show" to it with x.className = "show"; to make the snackbar visible. The class "show" will be removed after 3000ms to make the snackbar dissapear again with the code setTimeout(function(){ x.className = x.className.replace("show", ""); }, 3000);
